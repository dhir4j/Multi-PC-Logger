PC Logger

Tool for System Monitoring and Reporting

About Project


As digital communication continues to grow, secure messaging platforms are becoming increasingly important. This project aims to address this need by developing a Python-based tool to monitor and report system information for multiple PCs. The tool will collect data such as RAM, HDD, and OS version via a Python script and store it in a database through a Django server. A Django app will then display the PC information, while a dashboard will provide critical alerts. By defining the project's scope, choosing appropriate tools, and designing the system, this project can help enhance the security and privacy of digital communication.
